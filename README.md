# UpRace!
Top-Down race project. A game that I will try to create a circuit that you need to win against some (coming soon) bots.
A simple game, still not complete. But I want to create a cool gameplay, something better than all projects I've made.

## About The Project
This is one of my goals to this year to consolidate my knowledge in game logic and OpenGL, and other technologies.
There's only two projects before I go to 3D, and there is where I want to create my coolest games. But, anyway,
this project will help me to consolidate and creat portfolio for future opportunities. I hope you enjoy this project.

## How Compile
Is not complete yet, I want to create the CMakeLists.txt to make more portable, but as I only have Makefile, you can,
having installed and setup in your computer, run `make` and compile, but maybe it not work well in all environments.
Overall, you'll need: 
- GNU Make
- GNU GCC (at least with C++23)
- CMake (in the future)
